      操作マニュアル
      • 今日のミッション：登録した予定を確認し、終わったら「クリア」を押す。「砂時計へ転記」でログ欄へ移せる。
      • 追加・編集：予定は「固定」「ゆる固定」「自由」から選べる。嫌になりやすい予定は「ゆる固定」がおすすめ。
      • 頭から出すメモ：入力は砂時計ログ内で行い、確認・ミッション化は今日のミッション画面のメモ欄で行う。
      • Googleショートカット：カレンダーとGmailを別タブで開ける。個人ページは埋め込みではなくリンク式。
      • 砂時計ログ：集中・休憩・勉強など、時間の使い道を小瓶で見える化する。履歴には予定時間をもとに、千葉県最低賃金ベースで「給与・利益・損失」に分けて表示する。
      • 今日をしめる：達成数に関係なく、今日のログとして記録できる。完璧主義の首輪を外すためのボタン。
      • 直前操作を取り消す：クリア、削除、今日をしめるなどの直前操作を1回だけ巻き戻せる。押し間違い用の保険。

      ショートカット：Ctrl + Enter でミッション保存。

      データ管理

      このサイトのデータはブラウザ内に保存される。別のPCや別ブラウザには自動共有されない。

      GitHub同期（PC ⇄ スマホ）

      GitHub Pagesは表示用の静的ページなので、このHTMLからリポジトリへ直接保存するのではなく、GitHub Gistを同期用の小さな保管庫として使います。トークンはHTMLには書き込まず、この端末のブラウザ内にだけ保存します。

      公開中のHTMLファイルにトークンを直接書かないこと。ここへ入力した値は、この端末のlocalStorageにだけ保存されます。
      未設定 トークンを入れて「同期設定を保存」を押すと、手動保存または2分ごとの自動同期が使えます。
