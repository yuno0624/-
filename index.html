      操作マニュアル
      • 今日のミッション：登録した予定を確認し、終わったら「クリア」。
      • 砂時計ログ：集中・休憩・勉強など、時間の使い道を小瓶で見える化。
      • 習慣トラッカー：週表と月カレンダーで継続を確認。筋トレは回復日も見る。
      • 履歴：ミッション・砂時計・習慣をまとめて時間収支表示。
      データ管理

      このサイトのデータはブラウザ内に保存される。別PCや別ブラウザには自動共有されない。

      GitHub同期（任意）

      GitHub Gistを同期用の小さな保管庫として使う。トークンはこの端末のlocalStorageに保存。同期ファイルは、指定したファイル名のJSONだけを読み込みます。

      未設定トークンを入れて同期設定を保存。
      読み込み時に「# 同期用Gist」のような説明文ファイルをJSONとして読まないよう、指定した同期ファイル名と完全一致するJSONだけを読み込みます。
